# 🎧 FeelMusic

Reproductor de música que recomienda canciones según tu estado de ánimo, con seguimiento de tu historial emocional y un perfil musical inteligente basado en tus datos de uso.

**🔗 Demo en vivo:** https://phenomenal-cupcake-0997d5.netlify.app/

## ✨ Features

- **Recomendación por emoción**: selecciona cómo te sientes (feliz, triste, relajado, energético, nostálgico, romántico, entre otros) y recibe música acorde
- **Reproductor integrado**: cola de reproducción ("Up Next") sobre YouTube
- **Historial emocional**: registro de cómo te has sentido a lo largo del tiempo
- **Estadísticas personales**: panel con métricas de tu actividad y hábitos de escucha
- **Perfil musical inteligente**: análisis de tu perfil basado en tu historial de emociones y música escuchada
- **Autenticación de usuarios** vía Firebase Auth

## 🛠️ Stack técnico

- **Frontend**: HTML, CSS, JavaScript (vanilla)
- **Autenticación**: Firebase Auth
- **Base de datos / Backend**: [Supabase](https://supabase.com/) — persistencia del historial emocional, estadísticas y perfil de usuario
- **Reproducción de música**: YouTube Data API
- **Hosting**: Netlify

## 🚀 Cómo usarlo

1. Entra a la [demo en vivo](https://phenomenal-cupcake-0997d5.netlify.app/)
2. Crea una cuenta o inicia sesión
3. Selecciona cómo te sientes en este momento
4. Deja que FeelMusic arme tu playlist según tu emoción
5. Revisa tus estadísticas y tu historial emocional en tu perfil
